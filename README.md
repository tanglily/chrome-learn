# chrome-learn

## 插件骨架生成

1. 安装[Node.js](https://nodejs.org/)

1. 安装yeoman

  ```
  npm install yo -g
  ```
  
1. 安装[generator-chrome-extension](https://github.com/yeoman/generator-chrome-extension)

   ```
   npm install generator-chrome-extension -g
   ```
   
1. 生成插件骨架

   ```
   yo chrome-extension
   ```
